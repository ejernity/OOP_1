Instructions:

1) Command for compilation: make

2) You can run: make clean
    and all .o files and executable metrosimulation will be deleted.

3) You have to give two arguments after the name of executable file. First argument for the number of stations and second for the number of waggons. For example: ./metrosimulation 15 18
